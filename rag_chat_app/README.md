# RAG Chat Application

This project implements a chat interface using a Retrieval-Augmented Generation (RAG) system. It consists of a FastAPI backend and a Streamlit frontend.

## Project Structure

```
rag-chat-app
├── backend
│   ├── app.py               # Entry point for the FastAPI backend
│   ├── models
│   │   └── rag_model.py     # RAG model class for handling logic
│   ├── routes
│   │   └── chat.py          # Chat route for processing requests
│   ├── services
│   │   └── retrieval.py      # Retrieval logic for the RAG system
│   └── requirements.txt      # Backend dependencies
├── frontend
│   ├── app.py               # Entry point for the Streamlit frontend
│   └── requirements.txt      # Frontend dependencies
└── README.md                # Project documentation
```

## Setup Instructions

### Backend

1. Navigate to the `backend` directory:
   ```
   cd backend
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the FastAPI application:
   ```
   uvicorn app:app --reload
   ```

### Frontend

1. Navigate to the `frontend` directory:
   ```
   cd frontend
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the Streamlit application:
   ```
   streamlit run app.py
   ```

## Usage

- Open your web browser and go to `http://localhost:8501` to access the chat interface.
- Enter your queries in the chat input and receive responses generated by the RAG system.
